# 安装指南

本风格指南可作为 AI 代码助手插件安装，在编写中文文档时提供格式和排版建议。

## Claude Code

分两步安装：

```bash
# 添加 marketplace
/plugin marketplace add https://github.com/RightCapitalHQ/chinese-style-guide.git

# 安装插件
/plugin install chinese-style-guide@rightcapital-chinese-style-guide
```

## Gemini

```bash
skills install https://github.com/RightCapitalHQ/chinese-style-guide
```

## Codex

```bash
skill-installer install https://github.com/RightCapitalHQ/chinese-style-guide/tree/main/.claude/skills/chinese-style-guide
```

---

## 使用

安装后，当你编写或修改中文文档时，AI 会自动参考本风格指南。你也可以直接询问：

- 「帮我检查这段文字的排版」
- 「修正这段文字的格式」
- 「按照中文排版规范调整」
