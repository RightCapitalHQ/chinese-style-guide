# 中文写作排版快速参考卡片

本文件提供格式修正时的快速查阅。

## 空格规则速查

```
✅ 添加空格
汉字 + 英文：这是 English 文本
汉字 + 数字：共 100 个

❌ 不添加空格
汉字标点 + 英文：在《Book》中
格式化内容：这是**重点**内容
半角标点：100 美元/月
```

## 标点符号速查

### 中文标点（全角）

```
句号：。（不是 .）
逗号：，（不是 ,）
问号：？（不是 ?）
感叹号：！（不是 !）
冒号：：（不是 :）
分号：；（不是 ;）
引号：""（不是 ""）
书名号：《》
括号：（）
破折号：——（两个 em dash）
省略号：……（两个 …）
```

### 半角标点

```
分隔号：/
百分号：%
数学符号：+ - * /
时间冒号：9:05
```

## 括号选择决策树

```
括号内容是英文？
├── 是 → 被修饰内容也是英文？
│         ├── 是 → 使用半角 ()
│         └── 否 → 使用全角（）
└── 否 → 使用全角（）
```

示例：
- `RightCapital (a fintech company)` - 都是英文，半角
- `手机应用（app）` - 被修饰的是中文，全角

## 数字格式速查

| 场景 | 格式 | 示例 |
|------|------|------|
| < 10 | 中文数字 | 三个、五次 |
| ≥ 10 | 阿拉伯数字 | 100 个、50 次 |
| ≥ 1000 | 千位分隔 | 3,000,000 |
| 万/亿 | 可用阿拉伯 | 300 万 |
| 百/千 | 用汉字 | 三百、五千 |

## 日期时间速查

### 中文日期

```
✅ 正确格式
2020 年 3 月 31 日
2020-03-31

❌ 错误格式
03 月 31 日（不要补零）
2020/3/31（不用斜杠）
20 年 3 月（年份要 4 位）
```

### 时间

```
✅ 正确格式
9:05（使用半角冒号）
上午 9:05

❌ 错误格式
9：05（全角冒号错误）
09:05 am（12 小时制不补零）
```

## 电话号码速查

### 手机号（3-4-4）

```
✅ 139-1234-5678
✅ 139 1234 5678
❌ 1391-234-5678
❌ 13912345678
```

### 座机（含区号）

```
✅ 010 6123-4567
✅ (010) 6123-4567
❌ 01061234567
❌ (010)6123-4567
```

### 国际号码

```
✅ +86 10 6123-4567
❌ +86 010 6123-4567（区号不加 0）
```

### 400/800 号码（3-3-4）

```
✅ 400-123-4567
❌ 4001234567
```

## 英文规则速查

### 拼写

```
使用美式英语：
✅ color, center, canceled
❌ colour, centre, cancelled
```

### 牛津逗号

```
✅ A, B, and C（使用牛津逗号）
❌ A, B and C
```

### 破折号

```
Em dash（破折号）：—
- 左右不留空格
- 不用两个

En dash（区间）：–
- 用于数字区间：2018–2019
```

### 引号和撇号

```
引号：""（弯引号）
撇号：'（不是 '）

✅ It's a "test"
❌ It's a "test"
```

## 常见错误对照

| 错误 | 正确 | 规则 |
|------|------|------|
| `这是English文本` | `这是 English 文本` | 中英加空格 |
| `共100个` | `共 100 个` | 中数加空格 |
| `在《 Book 》中` | `在《Book》中` | 书名号内不加空格 |
| `请问,这是什么?` | `请问，这是什么？` | 用全角标点 |
| `北京-上海` | `北京—上海` | 用一字线 |
| `3000000` | `3,000,000` | 千位分隔符 |
| `03月31日` | `3 月 31 日` | 不补零+加空格 |
| `9：05` | `9:05` | 时间用半角冒号 |
| `13912345678` | `139-1234-5678` | 手机号分组 |
