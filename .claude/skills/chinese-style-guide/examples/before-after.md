# 格式修正前后对比示例

本文件展示常见格式问题的修正前后对比。

---

## 示例 1：技术文档

### 修正前

```
我们的API服务每秒可处理10000次请求,使用了Redis作为缓存层.

当用户调用getUserInfo()函数时,系统会先检查缓存,如果缓存中没有数据,则从MySQL数据库中查询.

技术栈包括:React,Node.js,和PostgreSQL.

更多信息请参考"The Complete Guide to API Design"这本书.
```

### 修正后

```
我们的 API 服务每秒可处理 10,000 次请求，使用了 Redis 作为缓存层。

当用户调用 `getUserInfo()` 函数时，系统会先检查缓存，如果缓存中没有数据，则从 MySQL 数据库中查询。

技术栈包括：React、Node.js 和 PostgreSQL。

更多信息请参考《The Complete Guide to API Design》这本书。
```

### 修正说明

| 问题 | 修正内容 |
|------|----------|
| 中英文无空格 | `API服务` → `API 服务` |
| 数字无千位分隔 | `10000` → `10,000` |
| 使用半角逗号和句号 | `,` → `，`，`.` → `。` |
| 函数名未格式化 | `getUserInfo()` → `` `getUserInfo()` `` |
| 并列词用逗号 | `React,Node.js` → `React、Node.js` |
| 牛津逗号错用 | `Node.js,和` → `Node.js 和` |
| 书名用引号 | `"..."` → `《...》` |

## 示例 2：商务邮件

### 修正前

```
尊敬的客户,

感谢您选择RightCapital的服务!我们的团队将在3月31日2024年与您联系.

如有疑问,请拨打我们的客服热线:4001234567或发送邮件至support@example.com.

此致
敬礼
```

### 修正后

```
尊敬的客户，

感谢您选择 RightCapital 的服务！我们的团队将在 2024 年 3 月 31 日与您联系。

如有疑问，请拨打我们的客服热线：400-123-4567 或发送邮件至 support@example.com。

此致
敬礼
```

### 修正说明

| 问题 | 修正内容 |
|------|----------|
| 半角逗号 | `,` → `，` |
| 公司名无空格 | `选择RightCapital` → `选择 RightCapital` |
| 半角感叹号 | `!` → `！` |
| 日期顺序错误 | `3月31日2024年` → `2024 年 3 月 31 日` |
| 半角冒号 | `:` → `：` |
| 电话号码未分组 | `4001234567` → `400-123-4567` |
| 邮箱地址无空格 | `至support` → `至 support` |

## 示例 3：产品说明

### 修正前

```
产品规格:

1.屏幕尺寸:27英寸
2.分辨率:5120x2880像素
3.存储容量:512GB SSD
4.重量:约6.3kg

价格:$2999.00(不含税)

该产品支持macOS,Windows,和Linux系统.
```

### 修正后

```
产品规格：

1. 屏幕尺寸：27 英寸；
2. 分辨率：5120×2880 像素；
3. 存储容量：512GB SSD；
4. 重量：约 6.3 kg。

价格：$2,999.00（不含税）

该产品支持 macOS、Windows 和 Linux 系统。
```

### 修正说明

| 问题 | 修正内容 |
|------|----------|
| 列表项编号后无空格 | `1.屏幕` → `1. 屏幕` |
| 半角冒号 | `:` → `：` |
| 数字与单位无空格 | `27英寸` → `27 英寸` |
| 乘号使用 x | `5120x2880` → `5120×2880` |
| 列表项结尾标点 | 添加分号和句号 |
| 价格无千位分隔 | `$2999` → `$2,999` |
| 半角括号 | `(不含税)` → `（不含税）` |
| 并列词用逗号 | `,Windows,和` → `、Windows 和` |

## 示例 4：中英混排的学术文档

### 修正前

```
在machine learning领域,deep learning(深度学习)已经成为主流方法.

根据Smith et al.(2023)的研究,使用transformer架构可以显著提升模型性能,准确率提升了15 %.

参考文献:
1."Attention Is All You Need"
2."BERT: Pre-training of Deep Bidirectional Transformers"
```

### 修正后

```
在 machine learning 领域，deep learning（深度学习）已经成为主流方法。

根据 Smith et al.（2023）的研究，使用 transformer 架构可以显著提升模型性能，准确率提升了 15%。

参考文献：
1. 《Attention Is All You Need》；
2. 《BERT: Pre-training of Deep Bidirectional Transformers》。
```

### 修正说明

| 问题 | 修正内容 |
|------|----------|
| 中英文无空格 | `在machine` → `在 machine` |
| 半角逗号和句号 | `,` → `，`，`.` → `。` |
| 括号选择 | `(深度学习)` → `（深度学习）` |
| 百分号前有空格 | `15 %` → `15%` |
| 论文用引号 | `"..."` → `《...》` |
| 列表项标点 | 添加分号和句号 |

## 示例 5：公司新闻稿

### 修正前

```
RightCapital(一家金融科技公司)今日宣布完成新一轮融资.

此次融资由ABC Capital领投,融资金额达5000万美元.

公司CEO表示:"这笔资金将用于扩大研发团队,目标是在2024年-2025年期间推出3个新产品."

联系方式:
电话:+8601061234567
邮箱:pr@rightcapital.com
```

### 修正后

```
RightCapital（一家金融科技公司）今日宣布完成新一轮融资。

此次融资由 ABC Capital 领投，融资金额达 5,000 万美元。

公司 CEO 表示："这笔资金将用于扩大研发团队，目标是在 2024 年—2025 年期间推出三个新产品。"

联系方式：
电话：+86 10 6123-4567
邮箱：pr@rightcapital.com
```

### 修正说明

| 问题 | 修正内容 |
|------|----------|
| 括号选择 | `RightCapital(...)` → `RightCapital（...）` |
| 中英文无空格 | `由ABC` → `由 ABC` |
| 数字格式 | `5000万` → `5,000 万` |
| 直引号 | `"..."` → `"..."` |
| 连接号 | `2024年-2025年` → `2024 年—2025 年` |
| 数字 < 10 | `3个` → `三个` |
| 电话号码格式 | `+8601061234567` → `+86 10 6123-4567` |

## 快速检查清单

修正文档时，按以下顺序检查：

1. [ ] **空格**：中英文、中数字之间是否有空格
2. [ ] **标点**：是否使用正确的全角/半角标点
3. [ ] **引号**：是否使用弯引号，书名是否用书名号
4. [ ] **括号**：根据内容选择全角或半角
5. [ ] **数字**：千位分隔、数字选择（中文/阿拉伯）
6. [ ] **日期**：年月日顺序、格式正确性
7. [ ] **电话**：正确分组
8. [ ] **代码**：技术术语是否格式化为代码
